## Crear proyecto con SpringBoot
https://start.spring.io/


## Ejecutar proyecto (sin compilar)
    gradle bootRun

### Cambiar puerto
    gradle bootRun --args='--server.port=8888'

### Ejecutar con .jar
    java -jar build/libs/sesion-4-0.0.1-SNAPSHOT.jar



## Compilar proyecto
    gradle clean build


### Consultar (GET)
http://localhost:8080/saludo
